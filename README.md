This repository contains a Jupyter Notebook (nltk.ipynb) that introduces the basics of Natural Language Processing (NLP) using the NLTK (Natural Language Toolkit) library in Python. It is beginner-friendly and focuses on core NLP concepts with hands-on examples.

## Topics Covered
- Introduction to Natural Language Processing (NLP)
- Word and sentence tokenization
- Stopwords removal
- Stemming and lemmatization
- Parts of Speech (POS) tagging
- Named Entity Recognition (NER)
- Working with NLTK corpora

## Technologies Used
- Python
- NLTK
- Jupyter Notebook

## Getting Started
1. Clone the repository  
git clone https://github.com/uttu4thecode/basics-of-nltk.git  
cd basics-of-nltk  

2. Install dependencies  
pip install nltk  

3. Download NLTK data  
Open Python or Jupyter and run:  
import nltk  
nltk.download('all')  

4. Run the notebook  
jupyter notebook  
Open nltk.ipynb
